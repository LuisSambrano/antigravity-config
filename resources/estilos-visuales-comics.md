# Biblioteca de Estilos Visuales para Cómics

Catálogo completo de estilos disponibles para generación de cómics con IA.

---

## 1. Manga (Estilo Japonés)

**Características**:

- Líneas limpias y expresivas
- Ojos grandes y expresivos
- Efectos de velocidad y movimiento
- Tonos de pantalla para sombras
- Viñetas dinámicas

**Mejor para**:

- Historias de acción
- Romance
- Fantasía
- Ciencia ficción

**Ejemplo de prompt**:

```
Estilo manga japonés con líneas limpias, ojos expresivos,
efectos de velocidad, y tonos de pantalla para sombras.
```

## 2. Americano Años 90

**Características**:

- Textura de papel vintage
- Colores saturados
- Líneas gruesas
- Sombreado con puntos (Ben-Day dots)
- Estética retro

**Mejor para**:

- Superhéroes
- Nostalgia
- Historias clásicas

**Ejemplo de prompt**:


## Cómo Elegir el Estilo Correcto

### Por Público Objetivo

**Niños (6-12 años)**:

- Looney Tunes
- Cartoon Network
- Chibi
- Pixel Art

**Jóvenes (13-25 años)**:

- Manga
- Superhéroes
- Cyberpunk
- Americano 90s

**Adultos (25+ años)**:

- Realista
- Noir
- Acuarela
- Steampunk

### Por Género de Historia

**Acción**: Manga, Superhéroes, Americano 90s  
**Comedia**: Looney Tunes, Chibi, Cartoon Network  
**Drama**: Realista, Acuarela, Noir  
**Ciencia Ficción**: Cyberpunk, Manga, Pixel Art  
**Fantasía**: Manga, Acuarela, Steampunk  
**Terror**: Horror, Noir  
**Romance**: Acuarela, Manga, Realista

## Combinaciones de Estilos

Puedes combinar estilos para crear algo único:

**Ejemplo 1**: Cyberpunk + Noir

```
Estilo cyberpunk noir en blanco y negro con neones brillantes,
alto contraste, y atmósfera misteriosa futurista.
```

**Ejemplo 2**: Manga + Acuarela

```
Estilo manga con técnica de acuarela, colores suaves y difuminados,
pero manteniendo ojos expresivos y líneas limpias.
```

**Ejemplo 3**: Superhéroes + Pixel Art

```
Estilo superhéroes retro con estética pixel art 16-bit,
poses dinámicas, y paleta limitada de colores vibrantes.
```

## Consejos para Mejores Resultados

1. **Sé específico**: Menciona elementos visuales concretos
2. **Usa referencias**: Nombra artistas o obras conocidas
3. **Define paleta**: Especifica colores dominantes
4. **Describe texturas**: Papel, digital, pintado, etc.
5. **Menciona iluminación**: Natural, dramática, suave, etc.
