# Antigravity Skills Inventory

Skills are contextual modules that are loaded directly by the AI agent based on the workspace requirements. There are no manual invocations.

## 1. Core
- Fundamental engineering protocols.
- Automated code reviews and repo audits.

## 2. Web
- `react`: Modern React 18/19 patterns (Server Components, Zustand, custom hooks).
- `nextjs`: Next.js App Router rules (Server Actions, caching, graceful degradation).
- `design-system`: Autonomous integration of UI/UX constraints. Parsers local `DESIGN.md` or extracts remote tokens from VoltAgent/awesome-design-md.
- `web-vitals`: Performance metrics (LCP < 2.5s) and Search Engine Optimization limits.

## 3. DB
- Base database modeling logic, row level security (RLS), and interactions.

## 4. AI
- Prompt architecture, agent orchestration frameworks.

## 5. Tools
- CLI, automations, Git hooks, deployment strategies.
